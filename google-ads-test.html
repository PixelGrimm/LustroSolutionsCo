Google Ads Tag Test Page

This page is used to test if the Google Ads tag is working correctly.

Instructions:

  1. Open your browser's Developer Tools (F12)
  2. Go to the Console tab
  3. Look for the message: "Google Ads tag loaded with ID: AW-17538915513"
  4. Check the Network tab for requests to googletagmanager.com
  5. Use Google Tag Assistant to verify the tag is firing
